Your bank's 11-character BIC code

Unique ID (max 35 chars)
Always pacs.008.001.08
Always swift.cbprplus.02

💳 2. Payment Information CdtTrfTxInf MANDATORY

Internal reference (max 16 chars)
Passed unchanged (max 35 chars)
Amount in currency selected below
INDA: settlement account held with beneficiary bank
SHA: charges split • OUR: sender pays all • BEN: receiver pays all
If payment goes through an intermediary

👤 3. Debtor (Payer) Dbtr MANDATORY

Bank holding the debtor's account

🏦 4. Creditor (Beneficiary) Cdtr MANDATORY

Beneficiary bank's BIC

📝 5. Remittance Information RmtInf OPTIONAL

Reference or description for the beneficiary (max 140 chars)
Structured reference like invoice number
Additional details for beneficiary

📋 6. Regulatory Reporting RgltryRptg OPTIONAL
